# CyberDiary 取扱説明書

## 1. このアプリについて
CyberDiary は、サイバーパンク風の見た目で日記を入力・保存・確認できるアプリです。

- 日記の作成
- AI風の分析メッセージ表示
- タグや登場人物のメタデータ管理
- 日記一覧の検索
- 詳細ページで内容を確認

---

## 2. まずは起動してみる
1. ターミナルを開く
2. このフォルダに移動する
3. 次のコマンドを実行する

```bash
Set-ExecutionPolicy RemoteSigned
```
確認メッセージが出たら: Y を押して「はい」と答えてください。これにより、ローカルで作成したスクリプトは実行可能になります。
```bash
cd "C:\Users\edgeo\Documents\コーディング\04_日記作成アプリ"
npm install
npm run dev
```

4. ブラウザで次のURLを開く

```text
http://localhost:3000
```

※ すでに別ポートで起動している場合は、表示されているURLを使ってください。
※ どのようなスクリプトでも実行可能な状態になっているので、このアプリを落とした場合は以下のスクリプトを打ってください。

```bash
Set-ExecutionPolicy Restricted
```

---

## 3. 使い方

### 日記を追加する
1. 「タイトル」にタイトルを入力
2. 「内容」に本文を入力
3. 必要なら「タグ（カンマ区切り）」を入力し、「キャラクタ」を選択
4. 「ログを保存」を押す

入力した内容は自動的に画面に反映されます。

### すでにある日記を見る
- 一覧に表示されたタイトルや「詳細」を押すと、詳細ページへ移動します。
- 詳細ページでは、本文・AI風の分析メッセージ・メタデータを確認できます。

### 日記を編集する
1. 一覧の「編集」を押す
2. 入力欄に内容を修正する
3. 「ログを更新」を押す

### 日記を削除する
- 一覧の「削除」を押すと、その日記を削除できます。

### 検索する
- 画面上部の検索欄に、タイトルやタグ、人物名を入力すると一覧を絞り込めます。

---

## 4. 日記データはどこに保存されるのか
このアプリは、日記データをサーバーのデータベースではなく、ブラウザの LocalStorage に保存しています。

つまり:
- 保存先は「あなたのブラウザ」内
- アプリ本体のサーバーにデータベースは設置していない
- 同じPCの同じブラウザであれば、ページを閉じても復元される

### 重要な注意
- ブラウザのキャッシュやサイトデータを削除すると、保存済みの日記も消えます
- 別のPCや別のブラウザでは、データは引き継がれません
- シークレットモードやブラウザの設定によっては保存の挙動が変わることがあります

---

## 5. 使うときのポイント
- まずはサンプルの日記を作って、保存・編集・削除の流れを試してみてください
- AIからのメッセージはOllama経由でローカルLLMが返します。
- モデルの既定値は `gemma4:12b` です（`OLLAMA_MODEL` 環境変数で上書き可能）。
- データは自分のブラウザに保存されるため、バックアップをしたい場合は手動でメモを残すのがおすすめです

### Ollama設定例

PowerShellで起動前に設定する場合:

```powershell
$env:OLLAMA_MODEL = "gemma4:12b"
npm run dev
```
